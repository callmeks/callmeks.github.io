# Vim Command for my references

## Command Count Motion
- **`8k`** = jump 8 line up
- **`8`** is referred as Count (can change to any number)
- **`k`** is referred as Motion (can change to others motion)
- **`d3j`** = delete current and 3 line below
- **`dw`** = delete a word
- **`y5j`** = yank 5 line below

## Normal mode
- motion
    - **`h`** = left
    - `**j**` = down
    - **`k`** = up
    - **`l`** = right
    - **`w`** = hops over by words
    - **`b`** = hops over by words (backwards)
- command
    - **`d`** = delete
    - **`u`** = undo
    - **`ctrl + r`** = redo
    - **`p`** = paste from yank or delete
    - **`y`** = yank / copy
    - **`v`** = visual mode or highlight

## Insert mode
- **`i`** = insert mode (left side on the cursor)
- **`esc`** = exit insert mode
- **`a`** = insert mode (right side on the cursor)
