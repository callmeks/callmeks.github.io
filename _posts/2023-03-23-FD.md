# Solution
- to print the flag, we will need 2 criteria
- first is comparing with the string **`LETMEWIN`**
- second is **`read()`** function
- to trigger **`read()`**, we will need to provide the correct number, **`fd`**
- **`fd`** is also known as File Descriptor which have specific meaning for each number
    - 0 = standard input
    - 1 = standard output
    - 2 = standard error
- Since we wants to insert a string, we will need to change the **`fd`** into 0
- **`atoi()`** is a function in C which converts strings into integer
- we will need to convert **`0x1234`** into integer to make **`fd`** to 0
- if the **`read()`** function is working, we will need to provide the needed string **`LETMEWIN`** to get the flag

## Solution.py
```python
from pwn import *

pwnlib.args.SILENT(info)
p = process(['/home/fd/fd',str(int(0x1234))])
p.sendline(b'LETMEWIN')
print(p.recvall())
p.close()
```
