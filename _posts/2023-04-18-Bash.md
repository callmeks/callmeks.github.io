## Stabilizing shell
```bash
python3 -c 'import pty;pty.spawn("/bin/bash");'
export TERM=xterm
[ctrl]+[z]
stty raw -echo;fg
stty columns 93 rows 44
```

## Escape Restricted shell
- When spacebar is restricted in a shell, make use of `${IFS}`

```bash
/var/root # echo "HI"
HI
/var/root # echo${IFS}"HI"
HI                          
```

## Add text to file with cat & EOF
- Useful when shell is limited
- EOF at the last to close syntax
```bash
/var/root # cat <<EOF >> test 
> test one
> test two
> EOF  

/var/root # cat test
test one
test two
```
