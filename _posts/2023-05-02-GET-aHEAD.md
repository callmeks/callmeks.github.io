## Description
```
Find the flag being held on this server to get ahead of the competition 
http://mercury.picoctf.net:45028/
```
## Solution
- According to the website, the 2 button there are used to change the background colour of the website.

- By having a look at their source code, we could see that they are using `get` and `post` method to change the colour of the background.
<br>
![](/assets/img/posts/2023-05-02-GET-aHEAD/2023-05-02-GET-aHEAD-20-06-52.png)
<br>
- Since there used different HTTP request method, we could also change the HTTP request method to something else to get flag.
- The title of the challenge, GET aHEAD, might be asking us to using HEAD request.
```bash
>> curl -I "http://mercury.picoctf.net:45028/index.php" 
HTTP/1.1 200 OK
flag: picoCTF{r3j3ct_th3_du4l1ty_775f2530}
Content-type: text/html; charset=UTF-8
```
- `curl -I` will send a HEAD request.
