# Source Code
```c
#include <dlfcn.h>
#include <fcntl.h>
#include <stdio.h>
#include <stdlib.h>
#include <unistd.h>

#define MAX_SIZE 65536

char bytes[MAX_SIZE];

void setup() {
    setvbuf(stdin, NULL, _IONBF, 0);
    setvbuf(stdout, NULL, _IONBF, 0);
    setvbuf(stderr, NULL, _IONBF, 0);
}

void shared(char *filename) {
    char *error;
    void *handle = dlopen(filename, RTLD_LAZY);
    if (!handle) {
        fputs(dlerror(), stdout);
        exit(1);
    }

    dlerror();
    void (*fptr)();
    if (dlsym(handle, "load")) {
        if (dlsym(handle, "my")) {
            if (dlsym(handle, "func")) {
                if ((fptr = dlsym(handle, "please"))) {
                    (*fptr)();
                }
            }
        }
    }
    if ((error = dlerror()) != NULL) {
        fprintf(stdout, "%s\n", error);
        exit(EXIT_FAILURE);
    }
    dlclose(handle);
}

int main(void) {
    setup();
    char filename[] = "/tmp/a.so";

    int fd = open(filename, O_CREAT | O_WRONLY, 0777);
    puts("Data: ");
    int nbytes = read(0, bytes, MAX_SIZE);
    if (nbytes < 0) {
        puts("read error");
        exit(EXIT_FAILURE);
    }
    write(fd, bytes, nbytes);
    close(fd);
    shared(filename);
    return 0;
}
```
# Solution
- This pwn challenge is about shared library.
- Based on the code given, it will create a temp file that writes in data and it will be used for **`shared()`** function.
- the temp file will be created as **`a.so`** file which is knowns as shared object file.
- In the **`shared()`** function, it is using **`dlopen()`** and **`dlsym()`** function.
- **`dlopen()`** function = open library
- **`dlsym()`** function = search for the specific function
- the **`shared()`** function is basically opening the created **`/tmp/a.so`** shared library and search for 4 function which are **`load`**, **`my`**, **`func`** and **`please`**. 
- It will then run the **`please()`** function
- Therefore the solution would be make use of **`please()`** function in shared library to print the flag.
- To create a **`a.so`** file, it requires a **`a.c`** file and convert to **`a.so`** file with **`gcc`**.
- Example of the **`a.c`** file

```c
#include <fcntl.h>
#include <unistd.h>
#include <stdio.h>
#include <stdlib.h>

void load(){}
void my(){}
void func(){}

void please()
{
    FILE *fptr;

    char c;
    // Open file
    fptr = fopen("./flag.txt", "r");
    if (fptr == NULL)
    {
        printf("Cannot open file \n");
        exit(0);
    }
  
    // Read contents from file
    c = fgetc(fptr);
    while (c != EOF)
    {
        printf ("%c", c);
        c = fgetc(fptr);
    }
  
    fclose(fptr);
}

```

- after that, compile with **`gcc`**

```bash
>> gcc a.c -shared -o a.so
```
- The **`a.so`** file will be the solution for this challenge.

```bash
cat a.so | ./sharing
```
